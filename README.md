<details><summary>instrucciones del profe</summary>
Para esta tarea deberán realizar un clasificador de imágenes que sea capaz de diferenciar entre 3 diferentes objetos. La idea es que tomen alrededor de entre 70-100 imágenes por objeto de 3 objetos cualesquiera para crear un dataset y utilizar una red neuronal capaz de procesar y clasificar con un alto grado de precisión los 3 objetos distintos. No se permitirán imágenes descargadas de internet. Lo mejor sería que crearan muchas muestras en diferentes escenarios de los objetos, incluso que contengan fondos diversos.

Deberán hacer un análisis de resultados para mostrar si el clasificador es apropiado para la tarea asignada. En otras palabras, generar una matriz de confusión y realizar un análisis completo de matriz de confusión. A su vez, mostrar la relación entre reales-Predichos y diversos ejemplos de imágenes etiquetadas por el algoritmo.

Posteriormente, cuando su clasificador muestre una alta precisión, deberán utilizar la cámara de su computadora y conectarla al modelo entrenado. De tal manera que, se pueda ver en tiempo real la clasificación del objeto a la que pertenece (pueden imprimir una etiqueta con el objeto identificado debajo de la cámara).
